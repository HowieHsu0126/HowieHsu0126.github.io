Traditional approaches cope with graphical data structures using a preprocessing phase which transforms the graphs into a set of ï¬‚at vectors. However, in this way, important topological information may be lost and the achieved results may heavily depend on the preprocessing stage. This paper presents a new neural model, called graph neural network (GNN), capable of directly processing graphs. A learning algorithm for GNNs is proposed and some experiments are discussed which assess the properties of the model.
